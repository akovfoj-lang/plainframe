# Archive

Cold storage for dropped ideas, superseded pages, and finished areas — kept for the record, routed to by nothing.

Rules:

- Nothing elsewhere in the repo is deleted in place; it moves here first (Never tier:
  deleting without an archive step). Deleting from the archive itself is a human
  decision, made rarely.
- Keep arrivals as-is. If the reason isn't obvious, add one line at the top of the moved
  file: when it landed here and why.
- /audit drops its dated reports here as `audit-YYYY-MM-DD.md`.
- /audit drafts a move of superseded `os/decisions.md` entries into
  `archive/decisions-superseded.md`. The ledger is a protected path (law 7), so the agent
  only drafts the move in the report; the owner enacts it.
- /audit drafts a yearly rotation of `os/worklog.md` into `archive/worklog-YYYY.md` once
  the log crosses into a new year, plus the pointer line the move adds to worklog.md's
  "Archived epochs" section (law 9 stays satisfiable after rotation) — same
  draft-then-owner-enacts pattern.
- MAP.md rolls this whole directory up to a single line, on purpose. The archive is for
  the record, not for the read path — nothing here should be needed to operate the
  system.
