# Worklog

Dated receipt lines for all work. Work isn't done until its receipt exists here (law 9).
If a session crashes between doing the work and writing the receipt, the work is treated
as not-done and repeated — never assumed complete from memory.

`os/scripts/gen-status.sh` surfaces the last 10 receipts from this file into `STATUS.md`.
Keep each receipt short: one line per unit of work, newest at the bottom.

Format: `YYYY-MM-DD <command-or-topic>: <one-line summary>`
