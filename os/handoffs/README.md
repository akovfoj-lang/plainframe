# Handoffs

The session handoff chain. Each file here is one session's parting note to the next.

- Write and name them per `os/playbooks/handoff.md` — the playbook owns the format and
  the `YYYY-MM-DD-HHMMSS-topic.md` naming rule (law 3: one home per fact).
- Never rewrite an old handoff. It is a record, not a page.
